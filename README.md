# Product-Metrics-Visualization-Food-Delivery
Дашборд демонстрирует состояние клиентской базы сервиса доставки еды.

## Ссылка на дашборд в Yandex DataLens 
**https://datalens.yandex/3aoly8gh3uban?tab=pA**  
На второй вкладке дашборда представлена аналитическая записка с выводами по проведенному исследованию и рекомендациями.

Детальные расчеты продуктовых метрик, отраженных на дашборде, представлены в файле **Product_Metrics_Food_Delivery.sql**


### Описание данных

Продукт для анализа — сервис доставки еды **«Всё.из.кафе»**.

Данные представлены пятью таблицами:

| Таблица | Описание |
|----------|----------|
| `analytics_events` | Журнал аналитических событий (логи пользователей) |
| `advertisement_budgets` | Рекламные затраты по дням и каналам привлечения |
| `partners` | Справочник партнёрских сетей и ресторанов |
| `dishes` | Справочник блюд |
| `cities` | Справочник городов |


#### Таблица `analytics_events`

Содержит данные о действиях пользователей в период с **30.04.2021** по **02.07.2021**.

| Поле | Описание |
|--------|----------|
| `visitor_uuid` | Идентификатор посетителя, присваиваемый каждому новому пользователю |
| `user_id` | Идентификатор зарегистрированного пользователя |
| `device_type` | Тип устройства пользователя |
| `city_id` | Идентификатор города пользователя |
| `age` | Возрастная группа пользователя |
| `source` | Рекламный источник привлечения |
| `first_date` | Дата первого посещения продукта |
| `visit_id` | Уникальный идентификатор сессии |
| `event` | Тип аналитического события |
| `datetime` | Дата и время события |
| `log_date` | Дата события |
| `rest_id` | Идентификатор сети ресторанов |
| `object_id` | Идентификатор блюда |
| `listing_id` | Идентификатор блюда в листинге рекомендаций |
| `position` | Позиция блюда в листинге |
| `order_id` | Идентификатор заказа |
| `revenue` | Выручка от заказа, ₽ |
| `delivery` | Стоимость доставки, ₽ |
| `commission` | Комиссия сервиса с выручки ресторана, ₽ |

##### Типы событий (`event`)

| Событие | Описание |
|----------|----------|
| `main_page` | Посещение главной страницы |
| `authorization` | Авторизация пользователя |
| `rest_page` | Просмотр страницы ресторана |
| `object_page` | Просмотр карточки блюда |
| `order` | Оплата заказа |


#### Таблица `advertisement_budgets`

Содержит данные о ежедневных рекламных расходах.

| Поле | Описание |
|--------|----------|
| `source` | Рекламный источник |
| `date` | Дата затрат |
| `budget` | Дневной рекламный бюджет, ₽ |


#### Таблица `partners`

Справочник партнёрских сетей ресторанов.

| Поле | Описание |
|--------|----------|
| `rest_id` | Идентификатор сети ресторанов |
| `chain` | Название сети |
| `type` | Тип кухни |
| `city_id` | Идентификатор города |
| `commission` | Комиссия сервиса с выручки ресторана, % |

> Одна сеть может быть представлена в нескольких городах.


#### Таблица `dishes`

Справочник блюд партнёрских ресторанов.

| Поле | Описание |
|--------|----------|
| `object_id` | Идентификатор блюда |
| `name` | Название блюда |
| `spicy` | Признак острого блюда (`1` — острое) |
| `fish` | Признак блюда с морепродуктами (`1` — содержит морепродукты) |
| `meat` | Признак мясного блюда (`1` — содержит мясо) |
| `rest_id` | Идентификатор ресторана |


#### Таблица `cities`

Справочник населённых пунктов, в которых доступен сервис.

| Поле | Описание |
|--------|----------|
| `city_id` | Уникальный идентификатор населённого пункта |
| `city_name` | Название населённого пункта |
